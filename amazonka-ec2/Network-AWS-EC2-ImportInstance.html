Description

Creates an import instance task using metadata from the specified disk image. After importing the image, you then upload it using the 'ec2-import-volume' command in the EC2 command line tools. For more information, see Using theCommand Line Tools to Import Your Virtual Machine to Amazon EC2 in the AmazonElastic Compute Cloud User Guide for Linux.

http://docs.aws.amazon.com/AWSEC2/latest/APIReference/ApiReference-query-ImportInstance.html
